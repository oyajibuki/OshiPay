How it works

応援が届くまで、わずか3ステップ

応援を受けたい人

1
新規登録・プロフィール生成

LINE・Google・Discordでボタン一つで登録完了。プロフィールを設定してあなた専用のQRコードを発行します。

2
QRをさりげなく公開

発行されたQRコードをファンが見える場所に公開して、応援を受け取ります。

※スタンプ応援は口座登録不要で今すぐ受け取れます。

3
口座登録（収益化）

メッセージ・応援金（投げ銭）を受け取りたい場合は、世界基準のセキュリティを持つStripe経由で安全に口座情報を登録します。累計¥10,000に達すると自動で銀行口座に振込まれます。

※手数料13.6%（OshiPay 10% + Stripe 3.6%）差引後の金額が振込されます。

応援したい人

QRコードをスキャン・かんたんログイン

専用アプリは不要。スマホの標準カメラで読み取り、LINE・Google・Discordでサッとログイン。

スタンプ／メッセージ／投げ銭で応援

Apple PayやGoogle Payで、わずか数秒でスマートに支払いが完了。温かい応援メッセージも一緒に送れます。
